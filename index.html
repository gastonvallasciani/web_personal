Últimos proyectos

Control de granja horticultura

Solución de IoT que permite automatizar una granja de horticultura. Mediante una página web embebida el usuario puede configurar el equipo. Posee un modo manual controlado por pulsadores y un modo automático por página web que permite configurar horarios de funcionamiento.

El equipo posee las siguientes características:

  • Selección de potencia de luces led entre 0 y 100%.
  • Función Amanecer y Atardecer.
  • Permite controlar etapa vegetativa y de floración de la planta.
  • Permite controlar una salida de 220v para controlar humedad del ambiente.

Contador de botellas

Solución de IoT que permite automatizar la recolección de botellas con aceite reciclado en puntos de recolección. Mediante un display color interactua con el usuario para el ingreso de botellas con aceite usado. Una vez ingresada la botella al contenedor premia al usuario con puntos para fomentar el reciclaje de aceite.

C FreeRtos WiFi 3G ESP32 Azure MQTT NTP

IOT-AVU

El IOT-AVU es un dispositivo que permite automatizar la recolección de aceite en puntos de recolección.

En cada punto de recolección se lleva a cabo la medición de nivel de aceite en un tanque y se envía la información recolectada a la plataforma de Azure.

Cuando en la plataforma se detecta que algún tanque esta lleno se programa una visita para recolectar el contenido del tanque.

La medición de nivel el tanque se puede realizar por medio de una sonda magnetostrictiva o un detector de 4 niveles. Mientras que la comunicación entre el dispositivo y la plataforma se puede llevar a cabo por medio de una conexión WiFi o 3G.

C FreeRtos WiFi 3G ESP32 Azure MQTT NTP RS485

WiPoint

El WiPoint es un equipo que permite la interconexión entre un servidor web y una consola de telemedición de tanques de combustible. Posee una página web embebida para realizar la configuración de parámetros de red y la conexión entre el WiPoint y el servidor web se realiza por medio de una conexión WiFi.
